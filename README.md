## 기능
1. 예금, 적금, 대출을 계산(Vue 사용)
2. 예금, 적금, 대출을 눌러도 사이트 주소가 안바뀌도록 Single Page Application으로 설계
3. 입력란에서 enter 입력시 계산
4. 들어올 수 없는 값 입력시 오류 출력
